# Number Guessing Game (C++)

## Description
This project is a console-based Number Guessing Game developed in C++ for the Programming Fundamentals course.

The program generates a random number between 1 and 10 and allows the user to guess it within a limited number of attempts. It also uses file handling to store and display previous game results.

---

## Features
- Random number generation  
- Maximum of 3 attempts  
- Hints provided (Too high / Too low)  
- Displays all guesses using arrays and pointers  
- Stores game results in a file  
- Displays previous results at program start  
- Simple and user-friendly menu  

---

## Technologies Used
- Programming Language: C++  
- Libraries: iostream, fstream, cstdlib, ctime  
- Platform: Windows / Linux  
- Interface: Console-based  

---

## How to Run
1. Open the project in Code::Blocks, Dev-C++, or Visual Studio  
2. Compile and run the source file  
3. Choose an option from the menu:
   - Press 1 to play the game  
   - Press 2 to exit  

---

## Game Rules
- The system generates a secret number between 1 and 10  
- The user has only 3 attempts to guess the correct number  
- The program provides hints after each guess  
- The result is saved in result.txt
